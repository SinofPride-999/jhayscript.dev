Getting Started with JhayScript

JhayScript is a modern programming language that combines the simplicity of BASIC with powerful features from contemporary languages. This guide will walk you through everything from basic syntax to advanced concepts.

hello.jhay
// Your first JhayScript program
function main() {
    print("Hello, World!")
}

Installation

To get started with JhayScript, download the appropriate package for your operating system:

# Using PowerShell
winget install JhayScript.JhayScript
# Verify installation
jhay --version
# Using Homebrew
brew install jhayscript
# Verify installation
jhay --version
# Debian/Ubuntu
sudo apt-get install jhayscript
# Verify installation
jhay --version

Variables & Data Types

JhayScript is strongly typed but supports type inference. Variables are declared with the initiate keyword.

Basic Types

variables.jhay
// Primitive types
initiate name = "Samuel" // string
initiate age = 21 // integer
initiate height = 1.75 // float
initiate isDeveloper = true // boolean

// Explicit type declaration
initiate counter: int = 0

Constants

Use const for values that shouldn't change:

variables.jhay
const PI = 3.14159
const MAX_USERS = 100

Type Conversion

variables.jhay
initiate numStr = "42"
initiate num = toInt(numStr) // Convert to integer
initiate strNum = toString(num) // Convert to string

Functions

Functions in JhayScript are declared with the function keyword and can have typed parameters and return values.

Basic Function

functions.jhay
function greet(name: string) -> string {
    return "Hello, " + name + "!"
}

initiate message = greet("Samuel")
print(message) // Output: Hello, Samuel!

Default Parameters

variables.jhay
function createUser(
    name: string,
    isAdmin: bool = false,
    permissions: [string] = []
) {
    // Function implementation
}

Higher-Order Functions

variables.jhay
function applyOperation(
    numbers: [int],
    operation: function(int) -> int
) -> [int] {
    return numbers.map(operation)
}

initiate squared = applyOperation([1, 2, 3], (x) => x * x)
print(squared) // Output: [1, 4, 9]

Control Flow

JhayScript provides familiar control flow structures with clean syntax.

Conditionals

conditionals.jhay
initiate age = 21

if age >= 18 {
    print("Adult")
} else {
    print("Minor")
}

// Ternary operator
initiate status = age >= 18 ? "Adult" : "Minor"

Loops

variables.jhay
// For loop
for i in 0..5 {
    print(i)
}

// While loop
initiate count = 0
while count < 5 {
    print(count)
    count += 1
}

Collections

JhayScript provides several collection types for managing groups of data.

Arrays

arrays.jhay
// Array creation
initiate numbers = [1, 2, 3, 4, 5]
initiate names: [string] = ["Alice", "Bob"]

// Accessing elements
initiate first = numbers[0] // 1
initiate last = numbers[-1] // 5

// Array methods
numbers.push(6) // Add to end
numbers.pop() // Remove from end
numbers.filter((x) => x > 3) // [4, 5]

Maps (Dictionaries)

variables.jhay
// Map creation
initiate user = {
    name: "Samuel",
    age: 21,
    isAdmin: true
}

// Accessing values
initiate userName = user.name // "Samuel"
initiate userAge = user["age"] // 21

Advanced Concepts

Classes and OOP

classes.jhay
class Person {
    private name: string
    private age: int

    constructor(name: string, age: int) {
        self.name = name
        self.age = age
    }

    method greet() -> string {
        return "Hello, my name is " + self.name
    }
}

initiate person = Person("Samuel", 21)
print(person.greet())

Async/Await

variables.jhay
async function fetchData(url: string) -> Result<string, string> {
    try {
        let response = await http.get(url)
        return Ok(response.text())
    } catch error {
        return Error("Failed to fetch data")
    }
}

Built-in Functions

print(value: any)

Outputs a value to the console.

len(collection)

Returns the length of an array or string.

range(start, end)

Generates a sequence of numbers.

toInt(str), toFloat(str)

Converts strings to numbers.

type(value)

Returns the type of a value.

readFile(path)

Reads a file asynchronously.
